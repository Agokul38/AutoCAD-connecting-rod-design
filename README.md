# AutoCAD-connecting-rod-design
3D Connecting Rod model designed using AutoCAD. Includes accurate mechanical dimensions, engineering drawings, and CAD documentation suitable for a mechanical engineering portfolio.
## Overview
This project showcases the design and 3D modeling of a Connecting Rod using AutoCAD. The model was created by following standard mechanical engineering design practices and includes accurate geometry suitable for educational, practice, and portfolio purposes.

## Features
- 3D Connecting Rod Model
- Precise Mechanical Dimensions
- Engineering Drawing
- AutoCAD DWG File
- PDF Drawing for Easy Viewing
